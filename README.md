# Overview
Scandium is a Vulkan-based game engine written in C++ that uses GLSL for shaders and GLM for mathematics. It began as a graphics programming learning project based on Brendan Galea's Vulkan engine tutorial, which I implemented from scratch while learning modern C++, Vulkan, and CMake.

Beyond the tutorial, I extended the engine by implementing additional 3D transformation utilities, including rotation and translation functions, and developed a seed generation system based on a pseudohash function for future procedural generation experiments. Building the project also required creating a cross-platform CMake build system targeting both Linux and Windows.

This project introduced me to low-level graphics programming and sparked my interest in computer graphics, which later led me to take COMP 557 (Computer Graphics) as an elective during my undergraduate studies.

## Features
- Vulkan-based rendering engine
- Cross-platform builds (Linux & Windows)
- CMake build system
- GLSL shader pipeline
- 3D object transformations
- Seed generation utilities for procedural content experiments

## Built With

- C++17
- Vulkan
- GLSL
